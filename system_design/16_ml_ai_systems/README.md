# 機械学習・AIシステム設計

## 概要

機械学習・AIシステムは、機械学習モデルを使用して推論、レコメンデーション、自然言語処理などを提供するシステムです。

## 主要システム

- **[Recommendation System](recommendation_system_design.md)** - レコメンデーションシステム
- **[ML Inference](ml_inference_design.md)** - 機械学習推論システム
- **[Chatbot](chatbot_design.md)** - チャットボットシステム

## 共通の設計課題

- **モデル推論**: 低レイテンシでの推論処理
- **特徴量エンジニアリング**: 特徴量の抽出と変換
- **A/Bテスト**: モデルの効果測定
- **モデル更新**: モデルの継続的な更新とデプロイ
- **スケーラビリティ**: 大量リクエストへの対応

## 学習の進め方

1. **Recommendation System**から始める: レコメンデーションの基本設計
2. **ML Inference**で推論システムを学ぶ: 大規模推論システム
