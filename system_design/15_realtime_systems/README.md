# リアルタイムシステム設計

## 概要

リアルタイムシステムは、低レイテンシでの双方向通信を可能にするシステムです。

## 主要システム

- **[Zoom](zoom_design.md)** - ビデオ会議プラットフォーム
- **[WebRTC](webrtc_design.md)** - リアルタイム通信プロトコル
- **[Realtime Gaming](realtime_gaming_design.md)** - リアルタイムゲーム

## 共通の設計課題

- **低レイテンシ**: ミリ秒単位での通信
- **メディア処理**: 音声・動画のエンコード・デコード
- **ピアツーピア通信**: 直接的な通信の確立
- **フォールバック**: 通信失敗時の代替手段
- **スケーラビリティ**: 大規模同時接続への対応

## 学習の進め方

1. **WebRTC**から始める: リアルタイム通信の基本プロトコル
2. **Zoom**でビデオ会議を学ぶ: 大規模ビデオ会議システム
