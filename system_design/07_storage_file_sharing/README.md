# ストレージ・ファイル共有システム設計

## 概要

ストレージ・ファイル共有プラットフォームは、ユーザーがファイルをアップロード、保存、共有するためのシステムです。

## 主要システム

- **[Dropbox](dropbox_design.md)** - ファイル同期・共有サービス
- **[Google Drive](google_drive_design.md)** - クラウドストレージサービス
- **[OneDrive](onedrive_design.md)** - Microsoftのクラウドストレージ
- **[iCloud](icloud_design.md)** - Appleのクラウドストレージ

## 共通の設計課題

- **ファイル同期**: 複数デバイス間でのファイル同期
- **バージョン管理**: ファイルの履歴管理と復元
- **共有機能**: ファイル・フォルダの共有と権限管理
- **ストレージ最適化**: 重複排除と圧縮
- **オフライン対応**: オフライン時のファイルアクセス

## 学習の進め方

1. **Dropbox**から始める: ファイル同期の基本設計
2. **Google Drive**でコラボレーションを学ぶ: リアルタイム共同編集
