# ホスティング・レンタルシステム設計

## 概要

ホスティング・レンタルプラットフォームは、宿泊施設や体験の予約を可能にするマーケットプレイスです。

## 主要システム

- **[Airbnb](airbnb_design.md)** - 宿泊施設レンタルプラットフォーム
- **[Booking.com](booking_design.md)** - ホテル予約プラットフォーム
- **[Expedia](expedia_design.md)** - 旅行予約プラットフォーム

## 共通の設計課題

- **検索とフィルタリング**: 条件に合った宿泊施設の検索
- **予約システム**: 在庫管理と予約処理
- **決済処理**: 安全な決済と返金処理
- **レビューシステム**: レビューと評価の管理
- **リアルタイム在庫**: リアルタイムでの在庫更新

## 学習の進め方

1. **Airbnb**から始める: マーケットプレイスの基本設計
2. **Booking.com**でホテル予約を学ぶ: 在庫管理システム
