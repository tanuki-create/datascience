# ニュース・メディアシステム設計

## 概要

ニュース・メディアプラットフォームは、ユーザーがコンテンツを投稿、閲覧、議論するためのシステムです。

## 主要システム

- **[Reddit](reddit_design.md)** - ソーシャルニュースアグリゲーション
- **[Medium](medium_design.md)** - ブログプラットフォーム
- **[Quora](quora_design.md)** - Q&Aプラットフォーム

## 共通の設計課題

- **コンテンツアグリゲーション**: 多様なソースからのコンテンツ収集
- **投票システム**: アップボート・ダウンボート機能
- **コメントスレッド**: ネストされたコメント構造
- **ランキングアルゴリズム**: 人気コンテンツのランキング
- **モデレーション**: 不適切なコンテンツの検出と削除

## 学習の進め方

1. **Reddit**から始める: コンテンツアグリゲーションの基本
2. **Medium**でブログプラットフォームを学ぶ: コンテンツ管理システム
