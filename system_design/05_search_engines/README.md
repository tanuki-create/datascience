# 検索エンジンシステム設計

## 概要

検索エンジンは、Web上の情報をインデックス化し、ユーザーのクエリに対して関連性の高い結果を返すシステムです。

## 主要システム

- **[Google検索](google_search_design.md)** - 世界最大の検索エンジン
- **[Bing](bing_design.md)** - Microsoftの検索エンジン
- **[DuckDuckGo](duckduckgo_design.md)** - プライバシー重視の検索エンジン

## 共通の設計課題

- **Webクローリング**: 数十億ページのクローリング
- **インデックス作成**: 大規模なインデックスの構築と更新
- **ランキングアルゴリズム**: 関連性の高い結果のランキング
- **低レイテンシ**: ミリ秒単位での検索結果返却
- **パーソナライゼーション**: ユーザーに適した結果の提供

## 学習の進め方

1. **Google検索**から始める: 大規模検索エンジンの包括的な設計
2. **DuckDuckGo**でプライバシーを学ぶ: プライバシー保護の設計
