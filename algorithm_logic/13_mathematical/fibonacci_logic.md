# Fibonacci - ロジック解説

## 問題概要

整数`n`が与えられたとき、n番目のフィボナッチ数を返す。

**例**:
```
Input: n = 5
Output: 5
説明: F(5) = F(4) + F(3) = 3 + 2 = 5
```

## ロジックの核心

### アルゴリズムのステップ

```
function fibonacci(n):
    if n <= 1:
        return n
    
    a, b = 0, 1
    for i in range(2, n + 1):
        a, b = b, a + b
    return b
```
