# Reverse Words in a String - ロジック解説

## 問題概要

文字列`s`が与えられたとき、単語の順序を反転させる。先頭と末尾の空白を削除し、単語間の複数の空白を1つに統一する。

**例**:
```
Input: s = "the sky is blue"
Output: "blue is sky the"
```

## ロジックの核心

### なぜ分割と結合が有効か？

**全探索（非効率）**:
- 文字列を1文字ずつ処理
- 時間計算量: O(n) - 線形時間だが実装が複雑

**分割と結合を使う理由**:
- **単語の抽出**: 空白で分割して単語を抽出
- **反転**: リストを反転
- **結合**: 空白で結合
- **時間計算量**: O(n) - 線形時間

### アルゴリズムのステップ

```
function reverseWords(s):
    words = s.split()
    words.reverse()
    return " ".join(words)
```

## 注意点と落とし穴

### 1. 空白の処理
- **問題**: 先頭・末尾・複数の空白を適切に処理する必要がある
- **解決策**: `split()`が自動的に処理
- **注意**: `split()`は複数の空白を1つに統一

### 2. 空の文字列
- **問題**: 空の文字列や空白のみの文字列の処理
- **解決策**: `split()`が空のリストを返す
- **注意**: エッジケースの処理が重要
