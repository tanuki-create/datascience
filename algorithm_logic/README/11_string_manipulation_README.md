# 文字列処理 (String Manipulation)

## 概要

文字列処理は、テキストデータを効率的に操作・検索するためのアルゴリズムです。パターンマッチング、文字列検索、圧縮などに応用されます。

## 基本概念

### 主要なアルゴリズム

1. **KMP（Knuth-Morris-Pratt）**: 効率的な文字列マッチング
2. **Trie（トライ木）**: 文字列の集合を効率的に管理
3. **Rabin-Karp**: ハッシュを使った文字列マッチング

## いつ使うべきか

- パターンマッチング
- 文字列検索
- プレフィックス/サフィックスの処理
- 文字列の圧縮
